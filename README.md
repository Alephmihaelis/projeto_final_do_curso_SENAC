# Projeto final do curso de Python-SENAC

Aplicativo web simples em que se faz o gerenciamento (exibição, adição, edição, exclusão) de uma lista de funcionários e de uma lista de recrutadores.

## Recursos utilizados

- HTML;
- CSS;
- JavaScript;
- BootStrap;
- Python;
- Flask;
- MySQL;
- Werkzeug.
